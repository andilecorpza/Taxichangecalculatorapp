# Taxi Change Calculator Pro

Taxi Change Calculator Pro is a simple Android app built with Python and Kivy.  
It helps taxi drivers and passengers quickly calculate change after a payment.

## Features

- Calculate taxi change instantly
- View history of transactions
- Simple and clean interface
- Transparent text boxes
- Background image support
- Designed for everyday taxi use

## Built With

- Python
- Kivy
- Buildozer
- Android SDK
